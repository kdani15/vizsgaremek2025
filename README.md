# Vizsgaremek 2025 - Filmkedvelő alkalmazás (Letterboxd inspirációval)

## Fő funkciók

- Regisztráció, bejelentkezés
- Filmek kedvelése
- Keresés cím alapján
- Filmoldal: poszter, leírás, kiadás éve, stb...
- Lejátszási lista

## Struktúra

- `backend/`: NestJS + PostgreSQL + TypeORM
- `frontend/`: ReactJS
- `db/`: ER-diagram, dump
